Дакументацыя да праграмы "Аналізатар выяў"
Апісанне

"Аналізатар выяў" - гэта праграма з графічным інтэрфейсам для аналізу тэхнічных характарыстык графічных файлаў. Праграма падтрымлівае асінхронную апрацоўку выяў, што дазваляе карыстальніку працаваць з інтэрфейсам падчас выканання аналізу.
# Функцыянальныя магчымасці
## Асноўныя функцыі:

    Аналіз тэхнічных параметраў выяў: памер, разрозненне, глыбіня колеру, тып сціску, фармат

    Падтрымка розных фарматаў: JPG, JPEG, GIF, TIF, TIFF, BMP, PNG, PCX

    Рэкурсіўны пошук: аўтаматычны пошук выяў у падпапках

    Асінхронная апрацоўка: магчымасць спыніць аналіз у любы момант

    Сарціроўка вынікаў: сарціроўка па любым слупку табліцы

    Падрабязная інфармацыя: дадатковая інфармацыя пра колерную сістэму

## Атрымоўваемая інфармацыя:

    Імя файла

    Памер у пікселях (шырыня × вышыня)

    Разрозненне (DPI)

    Глыбіня колеру (бітнасць)

    Метод сціску

    Фармат файла

    Колерная сістэма (дадаткова)

# Сістэмныя патрабаванні
## Мінімальныя патрабаванні:

    Аперацыйная сістэма: Windows 7/8/10/11, Linux з GUI, macOS

    Працэсар: 2-ядерны працэсар 1.5 GHz ці вышэй

    Аператыўная памяць: 2 GB RAM

    Вольнае месца на дыску: 100 MB

    Экран: мінімум 1024×768 пікселяў

# Рэкамендаваныя патрабаванні:

    Аперацыйная сістэма: Windows 10/11, сучасныя дыстрыбуты Linux, macOS 10.14+

    Працэсар: 4-ядерны працэсар 2.0 GHz ці вышэй

    Аператыўная памяць: 4 GB RAM ці больш

    Вольнае месца на дыску: 200 MB

    Экран: 1920×1080 пікселяў ці вышэй

# Неабходнае праграмнае забеспячэнне:

    Python 3.7 ці навейшай версіі

    Бібліятэкі Python:

        PyQt5 ≥ 5.15

        Pillow (PIL) ≥ 8.0

        asyncio (уваходзіць у стандартную бібліятэку Python 3.7+)
